# fzf_select_github_repository

fzf source to cd to repository in `$HOME/GitHub`.

## Prerequisites

- fzf

## Installation

```
$ fisher unreliable-angels/fzf_select_github_repository
```

## Example key binding

```
function fish_user_key_bindings
    ...

    bind \cg fzf_select_github_repository

    ...
end
```
